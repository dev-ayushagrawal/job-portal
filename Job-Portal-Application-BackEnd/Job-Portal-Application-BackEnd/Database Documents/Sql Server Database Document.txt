#Project Features

1] CRUD Operation
	- Job Templet [ Create , Read, Delete, Update ]
	- Job Application [ Create, Read , Delete ]
	- Feedback System [ Create , Read, Delete ]
2] Pagination System [ UserDashboard (Jobs Page) , AdminDashBoard (Jobs Page, Applicantion Page, Feedback Page)]
3] MasterTables
4] Role Based Authentications ( Login / Logout ) Note : Admin SignUp Not Possible Without master password
5] Flexible UI
6] RoleBased Controller (For Code Readeble)
7] One Page Application Implementation
8] Search / Filter Functinality On Stream and Field
9] Route Authentication System
